# Troubleshooting

Having issues with LangGuard? This section covers common problems and their solutions.

## Quick Checks

Before diving into specific issues, try these quick fixes:

1. **Refresh the page** - A simple reload often resolves temporary issues
2. **Clear browser cache** - Old cached data can cause problems
3. **Try incognito mode** - Rules out browser extension conflicts
4. **Check your network** - Ensure you have a stable internet connection

## Error Messages

### Common Errors

| Error | What It Means | What To Do |
|-------|---------------|------------|
| "Session expired" | You've been logged out | Log in again |
| "Unauthorized" | Not logged in or session ended | Log in again |
| "Forbidden" | You don't have permission | Contact your admin |
| "Connection failed" | Can't reach external service | Check integration credentials |
| "Rate limit exceeded" | Too many API requests | Wait a few minutes |

### Self-Service Tips

1. **Check integration status** - Go to Integrations page to see connection health
2. **Verify sync status** - Look at "Last Sync" time on integration cards
3. **Adjust filters** - Ensure time range and filters aren't hiding data
4. **Try a different browser** - Rule out browser-specific issues

### Contact Support

If you can't resolve the issue:

1. **Email us**: [info@langguard.ai](mailto:info@langguard.ai)
2. **Include helpful details**:
   - What you were trying to do
   - What happened instead
   - Any error messages you saw
   - Screenshots if applicable

## Browser Tips

### Clear Cache

**Chrome**: Settings → Privacy → Clear browsing data

**Firefox**: Settings → Privacy → Clear Data

**Safari**: Develop → Empty Caches

### Try Incognito Mode

- **Chrome**: Cmd/Ctrl + Shift + N
- **Firefox**: Cmd/Ctrl + Shift + P
- **Safari**: Cmd + Shift + N
