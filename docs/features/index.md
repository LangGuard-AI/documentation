# Features Overview

LangGuard provides a comprehensive suite of features for AI governance and observability. This section covers all the capabilities available in the platform.

## Core Capabilities

### Discovery & Inventory

Automatically discover and catalog all AI assets across your organization:

- **Agent Discovery** - Identify all AI agents from trace data
- **Model Tracking** - Track which models are in use
- **Tool Inventory** - Catalog all tools and APIs used by agents
- **Dependency Mapping** - Understand relationships between components

[Learn more about Discovery →](/features/discovery)

---

### Real-time Monitoring

Monitor your AI operations with comprehensive dashboards:

- **Health Metrics** - Success rates, error rates, latency
- **Cost Tracking** - Token usage and cost analysis
- **Volume Analytics** - Request patterns and trends
- **Alerting** - Get notified of issues (coming soon)

[Learn more about Monitoring →](/features/monitoring)

---

### Trace Explorer

Deep-dive into individual traces with powerful analysis tools:

- **Advanced Search** - Full-text search across all trace data
- **Rich Filtering** - Filter by status, agent, time, and custom attributes
- **Detail View** - Complete trace information including spans and metadata
- **Comparison** - Compare multiple traces side-by-side

[Learn more about Trace Explorer →](/features/trace-explorer)

---

### Agent Activity View

Visualize agent behavior with multiple visualization modes:

- **Timeline View** - See when traces occurred over time
- **Performance View** - Track success rates, latency, and costs
- **Tools View** - Heatmap of tool usage across agents
- **Graph View** - Dependency relationships and data flow

[Learn more about Agent Activity →](/features/agent-activity)

---

### Data Catalog

Browse and manage discovered entities:

- **Hierarchical View** - Navigate catalogs, schemas, and tables
- **Metadata Management** - View and edit entity metadata
- **Lineage Tracking** - Understand data dependencies
- **Search** - Find entities across your data estate

[Learn more about Data Catalog →](/features/data-catalog)
