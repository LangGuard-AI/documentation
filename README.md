# LangGuard Documentation

Public documentation site for [LangGuard](https://langguard.ai), an AI Governance & Observability Platform. Built with [Docusaurus 3](https://docusaurus.io/).

**Live site:** https://docs.langguard.ai

## Requirements

- Node.js 20+

## Installation

```bash
npm install
```

## Local Development

```bash
npm run start
```

Starts a local development server and opens a browser window. Most changes are reflected live without having to restart the server.

## Build

```bash
npm run build
```

Generates static content into the `build` directory.

## Serve Built Site

```bash
npm run serve
```

Serves the built site locally for testing.

## Other Commands

```bash
npm run typecheck  # Run TypeScript type checking
npm run clear      # Clear Docusaurus cache
```
